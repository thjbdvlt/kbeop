clavier BÉPO modifié.

toutes les touches sont montées de deux rangées pour placer sous les pouces les touches Return, Escape, BackSpace, Tab ainsi que les touches modificatrices (+ quelques aménagements).

```
kbéop   vdlz
àaueij wtsrnm
 qxy., g'chf

 BE     HT
NSR    MG
    C
```
- B: Backspace
- E: Escape
- N: Nombres (4)
- S: Space
- R: Return
- H: Hyper (Super, CMD, ...)
- T: Tab
- M: Maj (Shift) (2)
- G: siGnes (3)
- C: Control

second niveau:
```
KBÉOP   VDLZ
ÀAUEIJ WTSRNM
 QXY:; G"CHF
```

troisième niveau:
```

@$()^~ “-=/\”
  []?! `+#*
```
